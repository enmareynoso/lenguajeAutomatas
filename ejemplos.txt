Ejemplos:

Automata:
  Tipo: determinista
  Inicio: q0
  Aceptacion: {q2}
  Transiciones:
      q0 -> q1: 0
      q0 -> q0: 1
      q1 -> q2: 1
      q1 -> q1: 0
      q2 -> q2: 0, 1


Automata:
  Tipo: no_determinista
  Inicio: q0
  Aceptacion: {q3}
  Transiciones:
      q0 -> {q0,q1}: 0
      q0 -> q0: 1
      q1 -> {q2,q3}: 1
      q1 -> q1: 0
      q2 -> q3: 0
      q3 -> q3: 0, 1
